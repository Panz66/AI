# AI

//  Masuk ke folder Project

//  Install Virtual Environment
    python -m venv venv

//  Aktifkan Virtual Environment
    venv\Scripts\activate

//  Install Dependencies Dari Requirements.txt
    pip install -r requirements.txt

//  Jalankan Aplikasi
    streamlit run app.py

Aplikasi akan jalan di http://localhost:8501

Catatan Tentang Aplikasi
Program dijalankan dengan aplikasi python dengan bantuan framework streamlit untuk gui webnya, proses klasifikasi dijalankan dengan workflow sesuai tugas dan algoritma random forest
